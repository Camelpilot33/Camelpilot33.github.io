Inverse Kinematics


Inverse kinematics is a concept that deals with determining the joint angles necessary to achieve a certain position of the end object of a kinematic chain. The algorithm I implemented is Forward and Backward Reaching Inverse Kinematics (FABRIK), which iteratively converges to the target. While calculating the end of an object is easy given joint parameters, the inverse is difficult, and there is no guarantee that a solution exists, which is why iterative methods are more flexible.
