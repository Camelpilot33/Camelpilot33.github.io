3D Renderer

This project is a basic raytracer, programmed in pure (vanilla) JS. This raytracer employs a simplified rendering pipeline, casting rays from the camera through each pixel and tracing their intersections with virtual objects like spheres and planes. It catches key lighting, such as diffused and specular reflection. It can handle point source and directional lighting, along with shadows. The scene to the right is rendered in real time when you load the page.
