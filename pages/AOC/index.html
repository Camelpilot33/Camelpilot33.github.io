Advent of Code

Advent of Code is an annual series of Christmas-themed programming challenges with over 1 million participants. For every day in December up to the 25th, a two part challenge is released, proceedingly getting harder. The event was founded and is maintained by Eric Wastl.

Every challenge consists of two parts, each with a puzzle, example input, and input. The puzzle and example input is the same for everyone, but the input is randomized.

Every day a list of the first 100 people to complete each puzzle is released. Higher placing on the list correspond to a higher score. As of writing this, I have only gotten on this list once, #44 for both parts on day 21 2022. I started this in 2020, and got 18/50 complete, then in 2021 I got 34/50, and in 2022 I got 50/50. Unfortunately, I lost my solutions in 2020, so the repository only contains 12 solutions for 2020, 39 for 2021, and all 50 for 2022.
