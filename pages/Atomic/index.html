Atomic Chess

Atomic Chess is a variant of standard chess, where captures remove all surrounding pieces except for pawns. You can play the game on Lichess.org or Chess.com. Lichess has stronger players and less cheaters, while Chess.com has more players. There also exists a Discord Server.

AtomicElo
As Chess.com's variant server doesn't include functionality to graph rating, I created a program to graph it. The site is live here. I also ported a compressed version to an extension, and you can view both project's source code here.

FishTrap
I made a python script which tests a player's moves against the top engine moves, helping identify cheaters. You can view the source code here. It currently has NaN downloads.
